# CV HTML with Animations - Plan

## Information Gathered:
- User wants a CV with HTML, modern design, and cool animations
- Color theme: Black and Purple (hitam ungu)
- Required sections: Header/Foto, About/Biodata, Pengalaman Kerja, Pendidikan, Skills
- Additional sections: Portfolio, Kontak

## Plan:
Create a single HTML file with embedded CSS and JavaScript containing:

1. **HTML Structure:**
   - Navigation bar (sticky with smooth scroll)
   - Header/Hero section with profile photo
   - About section
   - Experience section (Pengalaman Kerja)
   - Education section (Pendidikan)
   - Skills section (with animated progress bars)
   - Portfolio section
   - Contact section
   - Footer

2. **Design Features (Black & Purple Theme):**
   - Dark background (#0a0a0a, #1a1a2e)
   - Purple accents (#8b5cf6, #a855f7, #c084fc)
   - Gradient effects
   - Glassmorphism cards
   - Modern typography (Poppins font)

3. **Animations:**
   - Scroll reveal animations (elements fade in on scroll)
   - Floating particles background
   - Hover effects on cards and buttons
   - Animated skill bars
   - Smooth scroll navigation
   - Pulsing profile photo border
   - Gradient text animations
   - Staggered reveal for list items

4. **Responsive Design:**
   - Mobile-friendly layout
   - Responsive grid system

## Files to Create:
- `index.html` - Complete CV with all features

## Followup Steps:
1. Create the HTML file with all sections
2. Add CSS styling with purple-black theme
3. Add JavaScript for animations
4. Test and verify the CV displays correctly
